GitHub Updates Summary

Initial Setup

Step 1: Clone Repository

        
        git clone https://github.com/yourusername/yourrepository.git
cd yourrepository

Step 2: Configure Git

        
        git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

Making Changes

Step 1: Check Status

        
        git status

Step 2: Add Changes

        
        git add .

Step 3: Commit Changes

        
        git commit -m "Describe your changes here"

Synchronizing with Remote

Step 1: Pull Latest Changes

        
        git pull origin main

Step 2: Push Changes

        
        git push origin main

Handling Branches

Step 1: Create a New Branch

        
        git checkout -b new-branch-name

Step 2: Switch Branch

        
        git checkout branch-name

Step 3: Merge Branch

        
        git checkout main
git pull origin main
git merge branch-name

Resolving Conflicts

Step 1: Edit Conflicted Files

Open the conflicted files in an editor and resolve the conflicts.

Step 2: Add and Commit Resolved Files

        
        git add .
git commit -m "Resolved merge conflict"

        cd /home/shschubert/PoetryCamera
